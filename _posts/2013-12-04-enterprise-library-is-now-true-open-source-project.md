<p>Although EntLib has long shared its source code (Patterns &amp; Practices where one of the first to do so) they are now accepting community contributions and can be declared as true OSS</p>

<ol>
	<li>
	<ol>
		<li>Starting Thursday, Nov 21, 2013, we will be accepting community contributions to the codebase (both new features and bug fixes), subject to the <a href="https://entlib.codeplex.com/wikipage?title=ContributingCode">contribution guidelines</a>.</li>
		<li>Microsoft patterns &amp; practices continues to staff the project to curate as well as engage in active development and sustained engineering together with the community.</li>
		<li>In the spirit of true open source, the p&amp;p project team will use the same process for making updates to the application blocks as any community member. No secret repositories, hidden issue trackers, or internal-only processes.</li>
		<li>Our quality bars are not lowered in any way.</li>
	</ol>
	</li>
</ol>
