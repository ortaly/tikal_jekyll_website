<p>It sounds ironical, but Microsoft really provides a great tool for web development supporting PHP.</p>
<p>&nbsp;</p>
<p>Welcome <a href="http://www.microsoft.com/web/webmatrix/">WebMatrix</a>!</p>
<p>&nbsp;</p>
<p>At first look it is yet another tool to support the native Microsoft .NET technology. In fact I don't remember that Microsoft ever supported non their stuff.</p>
<p>&nbsp;</p>
<p>But the world is changing and now WebMatrix is not only tool for .NET developing. With WebMatrix you can write a pure PHP web application! It is amazing! Isn't it?</p>
<p>&nbsp;</p>
<p>In this post will show you how to start a new PHP web application with WebMatrix.</p>
<p>&nbsp;</p>
<p>When you first time run WebMatrix you get a splash screen with four option of starting using WebMatrix.</p>
<p>&nbsp;</p>
<p><a href="https://picasaweb.google.com/lh/photo/K9LUodw9IoibL_CZeFwP1TgAANDkdsfZKq9zdjvyQwU?feat=embedwebsite"><img width="640" height="469" src="https://lh4.googleusercontent.com/_XoCEelnLy98/TV0WldMZ8bI/AAAAAAAALDo/e40gU9CiHf8/s640/2011-02-17%2012h45_23.png" alt="" /></a></p>
<p>&nbsp;</p>
<p>As new WebMatrix user I start with &quot;Site from template&quot; which takes me to selection of template.</p>
<p>&nbsp;</p>
<p><a href="https://picasaweb.google.com/lh/photo/qNoG488qpnDB3pMKsqgdGTgAANDkdsfZKq9zdjvyQwU?feat=embedwebsite"><img width="640" height="469" src="https://lh6.googleusercontent.com/_XoCEelnLy98/TV0XD0ZZYII/AAAAAAAALD8/to_iLAlaIC0/s640/2011-02-17%2012h45_51.png" alt="" /></a></p>
<p>&nbsp;</p>
<p>There you may chose any template from the list, but for keeping this simple I select &quot;Empty Site&quot;.</p>
<p>By clicking &quot;OK&quot; button WebMatrix create the site and now it is a time to take a look at the site settings.</p>
<p>In the settings you will find &quot;Enable PHP&quot; checkbox. That's the thing! It not only enables PHP, it lets you select which PHP version you want to use!</p>
<p>&nbsp;</p>
<p><a href="https://picasaweb.google.com/lh/photo/xMil-wds6GGKmm0sn5WuAjgAANDkdsfZKq9zdjvyQwU?feat=embedwebsite"><img width="640" height="469" src="https://lh6.googleusercontent.com/_XoCEelnLy98/TV0XEi4StxI/AAAAAAAALEA/JymAtPZoVN0/s640/2011-02-17%2012h46_43.png" alt="" /></a></p>
<p>&nbsp;</p>
<p><a href="https://picasaweb.google.com/lh/photo/YEj8kUy-7ldyFdoBTgrzCTgAANDkdsfZKq9zdjvyQwU?feat=embedwebsite"><img width="640" height="469" src="https://lh4.googleusercontent.com/_XoCEelnLy98/TV0XElzkTXI/AAAAAAAALEE/rkg0-LN4Sv4/s640/2011-02-17%2012h48_02.png" alt="" /></a></p>
<p>&nbsp;</p>
<p>After configuring PHP there is a time to start codding. Switch to to &quot;Files&quot; view and &quot;New&quot; button on the toolbar. It prompts you to select a file type (which is actually file template). In the list of file types pick &quot;PHP&quot; and name it &quot;index.pxp&quot;</p>
<p>&nbsp;</p>
<p><a href="https://picasaweb.google.com/lh/photo/FNvm9IaKrPvvUQAr-InWBDgAANDkdsfZKq9zdjvyQwU?feat=embedwebsite"><img width="640" height="469" src="https://lh5.googleusercontent.com/_XoCEelnLy98/TV0XExfMIoI/AAAAAAAALEI/noj3MkQM5CA/s640/2011-02-17%2012h56_00.png" alt="" /></a></p>
<p>&nbsp;</p>
<p>The new file is opened in the file editor. Lets start coding. I replace existing content with 'classic' phpinfo();. As you may see, they have PHP syntax highlighting there!</p>
<p>&nbsp;</p>
<p><a href="https://picasaweb.google.com/lh/photo/cpUiBd6Ku4cQ3EI9E_DwbjgAANDkdsfZKq9zdjvyQwU?feat=embedwebsite"><img width="640" height="469" src="https://lh4.googleusercontent.com/_XoCEelnLy98/TV0XFBvyAPI/AAAAAAAALEM/cSWzzLcYLq8/s640/2011-02-17%2013h00_08.png" alt="" /></a></p>
<p>&nbsp;</p>
<p>Lets run!</p>
<p>Switch back to &quot;Site&quot; view. There you may find a URL of web server (IIS 7 express) which hosts this site. Just click the link.</p>
